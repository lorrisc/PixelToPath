PNG 与 SVG：有什么区别？

🖼️ 什么是PNG文件？

PNG (便携式网络图形) 是一种光栅图像格式。它将图像存储为固定的像素网格。当您放大或以大尺寸打印PNG时，您会注意到图像变得模糊或像素化。PNG文件非常适合固定尺寸的照片、屏幕截图和Web图形——但它们不能缩放。

📐 什么是SVG文件？

SVG (可缩放矢量图形) 是一种矢量图像格式。它不使用像素，而是使用数学路径和坐标来描述形状。SVG标志可以以10像素或10米的尺寸显示，并保持完美的清晰度。SVG文件的体积也更小，可在代码中编辑，并可使用CSS设置样式——这使它们成为标志、图标和插图的理想选择。

🔄 什么是矢量化（光栅转矢量）？

矢量化是将光栅图像（如PNG）转换为矢量文件（如SVG）的过程。PixelToPath使用Potrace算法来追踪PNG的轮廓，并生成平滑、可缩放的SVG路径。当您需要在不降低质量的情况下调整标志大小，或为激光切割机或丝网印刷机准备图稿时，这至关重要。

⚡ 什么时候应该将PNG转换为SVG？

  • 为广告牌或大型印刷品调整标志大小
  • 为刻字机或激光雕刻机准备图稿
  • 为Web或移动应用程序转换图标集
  • 在Inkscape、Figma或Illustrator中编辑矢量路径
  • 减小文件大小以提高Web性能
  • 使用CSS或JavaScript为形状制作动画
