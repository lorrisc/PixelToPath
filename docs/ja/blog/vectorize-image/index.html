ベクター化とは？

ピクセルベースの画像を、数学的パスで作られたベクターグラフィックに変換するプロセスです。

ソフトウェアは色とエッジを分析し、SVG形式のオブジェクトとして再構築します。

ラスター vs ベクター

ラスター画像は拡大すると劣化します。

ベクター画像は数学的に形状を保存します。利点:

  • 無制限のスケール: いつでも鮮明
  • 編集可能: パスを操作できる
  • 小さいファイルサイズ: アイコンなどに最適
  • 機械対応: レーザーカッターなどに必要
