仕組み

セットアップもアカウントも不要。アップロード、プレビュー、ダウンロードが10秒以内に完了。

📁

1. アップロード

ドラッグ＆ドロップまたはクリックで PNG、JPG、BMP、WebP をアップロード。クリップボードからの貼り付けも可能です。

⚡

2. ライブプレビュー

設定を変更すると SVG が自動的に更新されます。変換ボタンをクリックする必要はありません。

💾

3. ダウンロード

SVG をダウンロードをクリックしてベクターファイルを取得します。Illustrator、Inkscape、Figmaで開けます。

PixelToPath は3つの最適化されたプリセットを提供します。パラメータを手動で調整することも可能です。

白黒 白黒

トレース前に画像をグレースケールに変換します。単色のクリーンなパスを生成します。最適:

  • ロゴとブランドマーク
  • アイコンとピクトグラム
  • 線画とスケッチ
  • カッティングシートとレーザー彫刻

ポスター フラットカラー

高い精度で複数のカラーレイヤーが重なり合ったものとしてトレースします。最適:

  • フラットなイラストとカートゥーン
  • インフォグラフィック
  • クリップアート
  • 明確な色を持つデジタルアート

写真 写真

実際の写真の複雑さを処理するために広範なグループ化とノイズフィルタリングを使用します。最適:

  • ポートレートと顔
  • 風景写真
  • 商品写真
  • グラデーションが多い複雑な画像

SVG で何ができますか？

PixelToPath で生成された SVG は標準的で、どこでも機能します。

🎨

Figma または Illustrator で編集

すべての図形は完全に編集可能なパスです。品質を落とさずに色を変更したりサイズを変更できます。

🌐

Web サイトで使用

<img> タグとして使用するか、CSS でスタイルを設定します。PNG より高速に読み込まれます。

🖨️

任意のサイズで印刷

名刺からビルボードまで、どんなサイズでもぼやけずに完璧な結果を得られます。

✂️

レーザーカット

LightBurn や Cricut Design Space に直接インポートできます。

💻

アプリと Web アイコン

React、Vue で完璧に機能します。

🎞️

CSS と JS のアニメーション

静的な画像をベクターアニメーションに変えます。

よくある質問

制限なしで本当に無料ですか？

はい、完全無料です。PixelToPath はオープンソース (MIT) プロジェクトです。

カラー画像に対応していますか？

はい。VTracer を使用しています。カラーには ポスター または 写真 を選択してください。

ファイルは保存されますか？

いいえ。処理後すぐに削除されます。

サポートされているフォーマットは何ですか？

PNG、JPG、JPEG、BMP、WebP (Ctrl+V も可)。

最大ファイルサイズは？

オンラインは 20 MB まで。無制限に使用するにはデスクトップアプリをダウンロードしてください。

白黒の結果がおかしいのはなぜですか？

詳細設定で 反転 をオンにしてください。

スプライン、ポリゴン、ピクセルの違いは何ですか？

スプライン は滑らかな曲線。ポリゴン は直線。ピクセル はピクセルアート用です。

商用プロジェクトで使用できますか？

はい、ライセンス制限はありません。

オフラインで無制限の変換が必要ですか？

デスクトップ版はコンピューター上で完全に動作し、インターネットは不要です。
