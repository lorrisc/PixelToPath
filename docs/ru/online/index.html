Как это работает

Без настроек и аккаунта. Загрузите, проверьте и скачайте — менее чем за 10 секунд.

📁

1. Загрузка

Перетащите или кликните, чтобы загрузить изображение PNG, JPG, BMP или WebP. Вставка из буфера обмена тоже работает.

⚡

2. Предпросмотр онлайн

SVG обновляется автоматически при изменении настроек. Все происходит мгновенно.

💾

3. Скачивание

Нажмите «Скачать SVG», чтобы получить векторный файл. Откройте в Illustrator, Inkscape, Figma.

Какой пресет выбрать?

PixelToPath предлагает три пресета. Вы также можете настроить каждый параметр вручную.

Ч/Б Черно-белое

Преобразует в градации серого. Идеально для одноцветных контуров. Подходит для:

  • Логотипов
  • Иконок и пиктограмм
  • Скетчей
  • Лазерной гравировки

Постер Плоский цвет

Трассировка с наложением слоев с высокой точностью. Подходит для:

  • Иллюстраций
  • Инфографики
  • Стикеров
  • Цифровых артов

Фото Фотографии

Более широкая группировка цветов для обработки сложных реальных фото. Подходит для:

  • Портретов
  • Пейзажей
  • Фото товаров
  • Сложных изображений с градиентами

Частые вопросы

Это действительно бесплатно и без ограничений?

Да, полностью бесплатно. PixelToPath — это Open Source проект (MIT).

Поддерживаются ли цветные изображения?

Да. Мы используем VTracer. Выберите Постер или Фото для цвета.

Сохраняются ли мои файлы?

Нет. Файлы удаляются сразу после обработки.

Какие форматы поддерживаются?

PNG, JPG, JPEG, BMP и WebP. Можно вставить из буфера обмена (Ctrl+V).

Какой максимальный размер файла?

В онлайне — до 20 МБ. Для больших файлов скачайте десктопное приложение.

Почему Ч/Б результат выглядит неправильно?

Включите переключатель Инвертировать в дополнительных настройках.

В чем разница между режимами Сплайн, Полигон и Пиксель?

Сплайн для плавных кривых. Полигон для прямых линий. Пиксель без сглаживания (для пиксель-арта).

Можно использовать SVG в коммерческих целях?

Да, без каких-либо лицензионных ограничений.

Нужна офлайн-конвертация без лимитов?

Десктопная версия работает локально и не требует интернета.
