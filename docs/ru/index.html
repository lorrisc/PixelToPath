Конвертируйте изображения в SVG
Бесплатно & Open Source

PixelToPath преобразует изображения PNG, JPG, BMP и WebP в масштабируемую векторную графику — с полной поддержкой цвета, предпросмотром и без водяных знаков.

Десктопное приложение
