什么是图像转 SVG 转换？

将基于像素的光栅图像转换为可缩放的矢量图形。

转换过程（矢量化）分析边缘和颜色，并将其重新创建为矢量路径。

为什么 SVG 是图形的首选格式

SVG 是徽标和图标的标准格式。

转换为 SVG 提供了光栅格式无法提供的功能：

  • 独立于分辨率：在屏幕和打印上完美呈现
  • 完全可编辑：在任何 SVG 编辑器中修改
  • 紧凑且网页友好：SVG 通常比 PNG 小
  • 机器兼容：激光切割机接受 SVG
