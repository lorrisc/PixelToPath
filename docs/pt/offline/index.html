PNG vs SVG: Qual é a Diferença?

🖼️ O que é um arquivo PNG?

PNG (Portable Network Graphics) é um formato de imagem rasterizada. Ele armazena imagens como uma grade fixa de pixels. Ao dar zoom ou imprimir um PNG em tamanho grande, você notará que a imagem fica desfocada ou pixelizada. Os arquivos PNG são ótimos para fotografias, capturas de tela e gráficos da web em tamanho fixo — mas eles não escalam.

📐 O que é um arquivo SVG?

SVG (Scalable Vector Graphics) é um formato de imagem vetorial. Em vez de pixels, ele descreve formas usando caminhos e coordenadas matemáticas. Um logotipo SVG pode ser exibido a 10px ou 10 metros com nitidez perfeita. Os arquivos SVG também têm tamanho menor, são editáveis em código e estilizáveis com CSS — o que os torna ideais para logotipos, ícones e ilustrações.

🔄 O que é vetorização (raster para vetor)?

Vetorização é o processo de converter uma imagem rasterizada (como um PNG) em um arquivo vetorial (como um SVG). O PixelToPath usa o algoritmo Potrace para traçar os contornos do seu PNG e gerar caminhos SVG suaves e escaláveis. Isso é essencial quando você precisa redimensionar um logotipo sem perder qualidade ou preparar artes para uma cortadora a laser ou impressora serigráfica.

O que é Potrace e por que é usado?

Potrace é um algoritmo de código aberto bem estabelecido e de alta qualidade para rastrear bitmaps em gráficos vetoriais. Ele produz caminhos limpos e suaves a partir de imagens rasterizadas em preto e branco e é o motor por trás de muitas ferramentas de vetorização profissionais. O PixelToPath fornece uma interface amigável sobre o Potrace para que qualquer pessoa possa usá-lo sem habilidades de linha de comando.
