বিনামূল্যে অনলাইন PNG থেকে SVG কনভার্টার

কীভাবে কাজ করে

কোনো সেটআপ নেই, কোনো অ্যাকাউন্ট নেই। আপলোড, প্রিভিউ, ডাউনলোড - ১০ সেকেন্ডের মধ্যে।

📁

১. আপলোড করুন

যেকোন PNG, JPG, BMP বা WebP আপলোড করতে ড্র্যাগ ও ড্রপ করুন। ক্লিপবোর্ড থেকে পেস্ট (Ctrl+V) কাজ করে।

⚡

২. লাইভ প্রিভিউ

সেটিংস পরিবর্তন করলে SVG স্বয়ংক্রিয়ভাবে আপডেট হয়। সব কিছুই ইনস্ট্যান্ট।

💾

৩. ডাউনলোড করুন

আপনার ভেক্টর ফাইল পেতে SVG ডাউনলোড-এ ক্লিক করুন। Figma, Illustrator-এ খুলুন।

আপনার কোন প্রিসেট ব্যবহার করা উচিত?

PixelToPath তিনটি অপ্টিমাইজড প্রিসেট অফার করে। আপনি ম্যানুয়ালিও অ্যাডজাস্ট করতে পারেন।

সাদাকালো ব্ল্যাক অ্যান্ড হোয়াইট

ট্রেস করার আগে ছবিটিকে গ্রেস্কেলে রূপান্তর করে। এর জন্য সেরা:

  • লোগো এবং ব্র্যান্ড
  • আইকন
  • স্কেচ
  • লেজার কাটিং

পোস্টার ফ্ল্যাট কালার

উচ্চ নির্ভুলতার সাথে স্ট্যাক করা স্তর হিসাবে ট্রেস করে। এর জন্য সেরা:

  • ইলাস্ট্রেশন
  • ইনফোগ্রাফিক্স
  • ক্লিপআর্ট
  • ডিজিটাল আর্ট

ছবি ফটোগ্রাফ

আসল ছবির জটিলতার জন্য নয়েজ ফিল্টারিং ব্যবহার করে। এর জন্য সেরা:

  • পোর্ট্রেট
  • ল্যান্ডস্কেপ
  • পণ্যের ছবি
  • গ্রেডিয়েন্ট সহ জটিল ছবি

আপনার SVG দিয়ে আপনি কী করতে পারেন?

ফাইলগুলো পুরোপুরি স্ট্যান্ডার্ড এবং সব জায়গায় কাজ করে।

🎨

Figma বা Illustrator-এ এডিট করুন

প্রতিটি আকার সম্পূর্ণ সম্পাদনাযোগ্য পথ।

🌐

ওয়েবসাইটে ব্যবহার করুন

<img> ট্যাগ হিসাবে ব্যবহার করুন বা CSS দিয়ে স্টাইল করুন।

🖨️

যেকোন আকারে প্রিন্ট করুন

বিজনেস কার্ড থেকে বিলবোর্ড পর্যন্ত, পিক্সেলেশন ছাড়াই।

✂️

লেজার কাটিং

LightBurn বা Cricut-এ সরাসরি ইম্পোর্ট করুন।

💻

অ্যাপ এবং ওয়েব আইকন

React, Vue-এর সাথে নিখুঁতভাবে কাজ করে।

🎞️

CSS ও JS অ্যানিমেশন

স্ট্যাটিক ছবিকে ভেক্টরে অ্যানিমেট করুন।

সচরাচর জিজ্ঞাস্য

এটি কি সত্যিই বিনামূল্যে এবং সীমাহীন?

হ্যাঁ। PixelToPath একটি ওপেন সোর্স প্রজেক্ট (MIT)।

এটি কি রঙিন ছবি সমর্থন করে?

হ্যাঁ। VTracer ব্যবহার করে। রঙের জন্য পোস্টার বা ছবি বেছে নিন।

আমার ফাইল কি সেভ করা হয়?

না। প্রসেস হওয়ার সাথে সাথেই মুছে ফেলা হয়।

কোন কোন ফরম্যাট সমর্থিত?

PNG, JPG, JPEG, BMP এবং WebP (এবং Ctrl+V)।

সর্বোচ্চ ফাইলের আকার কত?

অনলাইনে 20 MB। বড় ফাইলের জন্য ডেস্কটপ অ্যাপ ডাউনলোড করুন।

সাদাকালো রেজাল্ট ভুল দেখাচ্ছে কেন?

অ্যাডভান্সড সেটিংসে ইনভার্ট চালু করুন।

স্প্লাইন, পলিগন এবং পিক্সেলের মধ্যে পার্থক্য কী?

স্প্লাইন মসৃণ রেখার জন্য। পলিগন সোজা রেখার জন্য। পিক্সেল পিক্সেল আর্টের জন্য।

আমি কি কমার্শিয়াল প্রজেক্টে ব্যবহার করতে পারি?

হ্যাঁ, কোনো লাইসেন্স বিধিনিষেধ নেই।

অফলাইনে আনলিমিটেড কনভার্সন দরকার?

ডেস্কটপ সংস্করণ পুরোপুরি আপনার কম্পিউটারে চলে - ইন্টারনেটের প্রয়োজন নেই।

⬇ Windows-এর জন্য ডাউনলোড করুন ⬇ Linux-এর জন্য ডাউনলোড করুন

ওপেন সোর্স ডেভেলপমেন্ট সমর্থন করুন

PixelToPath ব্যবহার করা বিনামূল্যে। যদি এই টুলটি আপনার সময় বাঁচাতে সাহায্য করে থাকে, তবে আমাকে এক কাপ কফি খাওয়ানোর কথা বিবেচনা করতে পারেন।
